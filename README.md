## Angular - Express - Vue

### Angular - 构建前台展示页面

### Express - 构建服务

### Vue - 构建后台管理页面

### 此项目是基于 Express 的后端服务

```bash
  // 安装依赖包
  npm install

  // 启动应用
  set DEBUG=blog_aev_e & npm start
```
  该项目是个博客系统；采用接口的方式，数据通过 Ajax 传输；数据存储采用 Mysql

  发布文章采用 Markdown 的编写方式，在后端通过 Marked 插件，把 Markdown 格式转成 HTML 格式
